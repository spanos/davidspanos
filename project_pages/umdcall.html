Urban Ministries of Durham Info System

The project

HackDuke 2014: Code for Good. Over the 24-hour hackathon, two groupmates and I created a tool for homeless shelters to keep track of bed occupancy and client information. Clients, meanwhile, can call to receive customized notifications from the shelter. We made the web app with HTML/CSS/jQuery, a PHP and MySQL backend, and the Twilio API.
